## 开始

```
npm install
```

## 目录结构

```
├── README.md
├── alipay.sql * 数据库转存文件
├── alipay_config.js *AlipaySDK 配置文件
├── checkSign.js  * 用于验签
├── createOrder.js * 创建订单并返回
├── mysql.js * 数据库配置
├── node_modules
├── package.json
├── public * 前端静态资源
├── readme.mda'sa
├── pem  * 存放应用私钥和应用公钥
└── server.js * 服务器文件，包含各种请求接口，可监听 http 服务
```
